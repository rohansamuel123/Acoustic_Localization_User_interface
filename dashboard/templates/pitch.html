The Problem Space

THE BATTLEFIELD
HAS BLIND SPOTS

Existing defense surveillance relies on optical, thermal, and radar systems — each constrained by physics, cost, and terrain. Sound travels where cameras cannot see.

LIMITATION 1
👁

Line-of-Sight Lock

Optical and thermal imaging systems require direct visual access. Walls, fog, foliage, and terrain breaks create dead zones where threats go undetected.

LIMITATION 2
💸

Prohibitive Cost

Radar systems and thermal arrays cost tens of thousands per unit — making dense sensor deployment financially impossible for many forward-operating scenarios.

LIMITATION 3
🌫

Environmental Degradation

Low-visibility conditions — smoke, dust, night — drastically reduce optical effectiveness. Radar requires power infrastructure. Neither is truly passive.

The Gap: Acoustic signals propagate omnidirectionally. They penetrate obstructions. They require no illumination, no line of sight, no radar signature. Sound carries position data — we just need to decode it.

Technical Core

HOW TDOA
WORKS

1

Sound Event Occurs

A sudden impulsive acoustic event — gunshot, explosion, balloon pop — radiates outward in all directions as a pressure wave traveling at 343 m/s.

2

Microphones Record Arrival Times

M1, M2, M3 each log the precise timestamp when they detect the sound peak. Since distances differ, arrival times t₁, t₂, t₃ are slightly staggered.

3

Compute TDOA Values

Pairwise differences: Δt₂₁ = t₂ − t₁, Δt₃₁ = t₃ − t₁, Δt₃₂ = t₃ − t₂. These encode distance-difference constraints as hyperbolic curves in space.

4

Hyperbola Intersection → Position

Each TDOA value defines a hyperbola of possible source locations. The intersection of three hyperbolas pinpoints the exact source coordinate (x, y).

Δt₂₁ = t₂ − t₁
Δt₃₁ = t₃ − t₁
Δt₃₂ = t₃ − t₂
→ Source @ (0.7, 0.6)

Process Architecture

8-STAGE
METHODOLOGY

01
Audio Acquisition

Microphones sample continuously, capturing ambient audio at all times.

02
Event Detection

Amplitude threshold triggers on sudden impulsive sounds (shots, blasts).

03
ToA Estimation

Precise time of arrival calculated at every microphone for each event.

04
TDOA Calculation

Pairwise time differences between all microphone pairs are computed.

05
Source Localization

Least-squares multilateration resolves the TDOA system to (x, y) coordinates.

06
Visualization

Position displayed live on a 2D map or coordinate grid in real time.

07
Data Logging

All events, TDOA values, and coordinates stored for post-analysis.

08
System Evaluation

Known-position test cases score accuracy; performance metrics recorded.
